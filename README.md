# RaktoSetu

RaktoSetu is a Location-Based Blood Donation and Emergency Matching Platform developed using the MERN Stack.

## Features

- Blood Donor Registration
- Emergency Blood Request
- Location Based Search
- Donor Availability Status
- Authentication & Authorization
- Admin Dashboard
- Donation Tracking

## Technology Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
